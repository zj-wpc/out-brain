# 欢迎来到外脑

这是「外脑」——所有 AI 输出物的存档。

## 分类说明

推送文档时可以指定以下分类：

| 分类 | 说明 |
|------|------|
| `research` | 调研报告、研究资料 |
| `code` | 代码片段、技术文档 |
| `idea` | 想法、脑暴记录 |
| `review` | 复盘、总结 |
| `news` | 新闻相关（与 news-briefing 联动） |
| `meta` | 关于本站本身 |

未指定分类时，系统会根据内容关键词自动判断。

## 发布方式

在 `news_briefing.py` 中，添加 `--jekyll --push --repo out-brain` 参数即可推送到本站。

或者手动推送 Markdown 文件到 `_posts/` 目录，文件名前缀格式：`YYYY-MM-DD-标题.md`
