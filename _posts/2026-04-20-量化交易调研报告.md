## 总结：技术栈全景

```
数据层  → AKShare / TuShare / Tushare Pro
回测层  → RQAlpha / Backtrader / 自研
策略层  → 因子研究 / 机器学习 / 深度强化学习
执行层  → VNPY / EasyTrader / 掘金SDK / QMT
云平台  → 聚宽 / 米筐 / 优矿 / 掘金
AI融合  → FinRL / FinHack / QuantMuse
```

**核心结论：**

- **VNPY** 是中国大陆影响力最大的开源量化框架，生态最完整（39.7k Stars）
- **AKShare** 是数据获取的事实标准（18.5k Stars）
- 商业平台（聚宽/米筐）聚合了数据+回测+社区，降低入门门槛
- AI 量化是明显趋势，FinRL/FinHack 代表了从传统因子模型向 RL/深度学习迁移的方向
